## QUICK START

### Prerequisites

- [Docker](https://docs.docker.com/get-docker/) + [Docker Compose](https://docs.docker.com/compose/install/)
- A folder full of video files
- (Optional) A [TMDB API token](https://www.themoviedb.org/settings/api) for metadata

### 1 вҖ” Clone & configure

```bash
git clone https://github.com/nemesbak/pixelvault.git
cd pixelvault

cp .env.example .env
# Edit .env:
#   MEDIA_PATH=/absolute/path/to/your/videos
#   TMDB_TOKEN=your_tmdb_token
```

### 2 вҖ” Launch

```bash
docker compose up -d
```

Open **http://localhost:5173**, register your account, and the library scan starts automatically.

### 3 вҖ” Connect Android

1. Download the APK from [Releases](https://github.com/nemesbak/pixelvault/releases)
2. Install it (allow "Unknown sources" if prompted)
3. Enter your server URL: `http://YOUR_SERVER_IP:3000`
4. Login or go to Settings вҶ’ **Pair Device** on the web UI to get a 6-digit code / QR

## DESIGN PHILOSOPHY

- **Pixel-art everywhere** вҖ” Press Start 2P font, `#39FF14` neon green, `#0a0a0a` background. Web and Android share the same visual language. No CSS frameworks, no UI component libraries.
- **Zero tracking** вҖ” no analytics, no telemetry, no cloud accounts required.
- **Direct play first** вҖ” the browser and ExoPlayer play the source file directly. Transcoding only kicks in for incompatible audio codecs (AC3/DTS вҶ’ AAC).
- **One command** вҖ” `docker compose up -d` starts everything.
